<p align="center">
  <b>BankFlow</b> simula as operações essenciais de um banco digital — contas, depósitos, saques,
  transferências e extrato. 🚀
</p>



---

## 🧩 Funcionalidades

| Funcionalidade         | Descrição                                              | Status |
|-------------------------|--------------------------------------------------------|:------:|
| 👤 Cadastro de conta     | Criação de conta com nome, CPF e senha                 |   ✅   |
| 💰 Depósito              | Adiciona saldo com animação de confirmação             |   ✅   |
| 💸 Saque                 | Retira saldo com validação de limite                   |   ✅   |
| 🔄 Transferência         | Envia dinheiro entre contas cadastradas                |   ✅   |
| 📄 Extrato               | Histórico de transações com data e valor               |   ✅   |
---

## 🛠️ Tecnologias utilizadas

- ☕ **Java 17+**
- 🧱 **POO (Programação Orientada a Objetos)**
- 📦 **Coleções (List, Map)** para gerenciamento de contas
